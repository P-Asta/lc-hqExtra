# HqExtra

A mod by Asta that helps newer-version settings stay usable on older versions of Lethal Company.

The generated BepInEx config path uses `asta.HqExtra.cfg`. Adjust any existing references if you were previously using the older plugin name.

Some of this mod's implementation was made with reference to the code from [RemoveMotionSway](https://thunderstore.io/c/lethal-company/p/kz/RemoveMotionSway/).

Report any issues on the issues page at [https://github.com/P-Asta/lc-hqExtra/issues](https://github.com/P-Asta/lc-hqExtra/issues)! Feel free to make pull requests for any other features or contributions. Thanks for checking out my work :).
